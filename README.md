## Smashup android-native

(Неофициальное) приложение-плеер для прослушивания мэшапов с ресурса smashup.ru.
Стэк технологий:
- Jetpack Compose
- архитектура MVVM
- Coroutines & Flow
- Android Media & Exoplayer
- Dagger-Hilt
- Clean Architecture
- Compose Navigation & Accompanist Navigation Animation
- Retrofit, OKHTTP, Glide

Версия 0.2.4
